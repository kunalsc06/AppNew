## ✨ Features
- Live weather search
- Current weather with detailed info
- 5-day forecast
- °C / °F toggle
- Dynamic background based on weather
- Fully responsive (mobile + desktop)
- Beautiful glassmorphism UI

## 🛠️ Tech Stack
- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
